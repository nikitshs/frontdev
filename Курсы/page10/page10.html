10. Публикация страницы в сети интернет

Существует множество различных способов опубликовать свой сайт в сети интернет. Но мы покажем один из бесплатных способов, с помощью платформы GitHub. Для начала нам нужно зарегистрироваться на данном сайте. Затем на панеле слева сверху мы нажимаем кнопку New.

Далее мы задаём имя нашему репозиторию и обязательно ставим галочку add a readme file и нажимаем зелёную кнопку снизу create repository.

Затем нажимаем кнопку Add file. В появившемся окне выбираем upload files.

Добавляем файлы своего сайта и жмём commit changes.

Далее переходим в настройки, затем раздел pages, там в разделе branch мы меняем none на main и нажимаем save.

Перезагружаем страницу и получаем ссылку на опубликованный сайт. Поздравляем, вы смогли опубликовать сайт!
