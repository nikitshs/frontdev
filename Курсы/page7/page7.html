7. CSS анимации

CSS-анимации позволяют анимировать переходы от одной конфигурации CSS стилей к другой. CSS-анимации состоят из двух компонентов: стилевое описание анимации и набор ключевых кадров, определяющих начальное, конечное и, возможно, промежуточное состояние анимируемых стилей.

Есть три преимущества CSS-анимации перед традиционными способами:
1.    Простота использования для простых анимаций; вы можете создать анимацию, не зная JavaScript.
2. Анимации будут хорошо работать даже при умеренных нагрузках системы. Простые анимации на JavaScript, если они плохо написаны, часто выполняются плохо. Движок может использовать frame-skipping и другие техники, чтобы сохранить производительность на таком высоком уровне.
3. Позволяет браузеру контролировать последовательность анимации, тем самым оптимизируя производительность и эффективность браузера. Например, уменьшая частоту обновления кадров анимации в непросматриваемых в данный момент вкладках.

Конфигурирование анимации

Чтобы создать CSS-анимацию вы должны добавить в стиль элемента, который хотите анимировать, свойство animation или его подсвойства. Это позволит вам настроить ускорение и продолжительность анимации, а также другие детали того, как анимация должна протекать. Это не поможет вам настроить внешний вид анимации, который настраивается с помощью @keyframes. Свойство animation имеет следующие подсвойства:

animation-name
Определяет имя @keyframes, настраивающего кадры анимации.

animation-duration
Определяет время, в течение которого должен пройти один цикл анимации.

animation-timing-function
Настраивает ускорение анимации.

animation-delay
Настраивает задержку между временем загрузки элемента и временем начала анимации.

animation-iteration-count
Определяет количество повторений анимации; вы можете использовать значение infinite для бесконечного повторения анимации.

animation-direction
Даёт возможность при каждом повторе анимации идти по альтернативному пути, либо сбросить все значения и повторить анимацию.

animation-fill-mode
Настраивает значения, используемые анимацией, до и после исполнения.

animation-play-state
Позволяет приостановить и возобновить анимацию.

В данном коде создаются три анимированные кнопки, которые перемещаются горизонтально по экрану. Каждая кнопка имеет свой класс и анимацию, что позволяет им двигаться с задержкой:

<button class="animated-button button-1"> Наведи на меня 1 </button>
<button class="animated-button button-2"> Наведи на меня 2 </button>
<button class="animated-button button-3"> Наведи на меня 3 </button>

Каждая кнопка имеет класс animated-button, который отвечает за общие стили, и уникальный класс button-1, button-2, button-3, позволяющий применять индивидуальные стили и анимацию:

.animated-button {
     padding: 15px 30px; - позволяет сделать внутренние отступы в кнопке
     background-color: #3498db; - задает цвет заднего фона
     color: white; - задает цвет текста
     border: none; - убирает границы
     border-radius: 10px; - сглаживает края
     cursor: pointer; - делает при наведении, чтобы мышка наводилась
     position: absolute; - задает абсолютную позицию над всеми элементами
     animation: move 4s linear infinite alternate; - делает анимацию длиной в 4 секунды, которая идет бесконечно
     transition: all 0.3s; - задает переход всем элементам в 0.3 секунды
}

@keyframes move {
     0% { left: 0; }
     100% { left: calc(100% - 265px); } - делает анимацию от левого края до правого края + -265 пикселей
}

Каждая кнопка также имеет animation-delay, что позволяет им двигаться с различной задержкой, создавая эффект «лестницы»:

.button-1 {
    animation-delay: 0.5s;
    font-family: "Montserrat", sans-serif;
    font-size: 20px;
    font-weight: bold;
    top: 140px;
    animation-delay: 1s;
}

.button-2 {
    animation-delay: 0.5s;
    font-family: "Montserrat", sans-serif;
    font-size: 20px;
    font-weight: bold;
    top: 140px;
    animation-delay: 1s;
}

.button-3 {
    animation-delay: 0.5s;
    font-family: "Montserrat", sans-serif;
    font-size: 20px;
    font-weight: bold;
    top: 140px;
    animation-delay: 1s;
}

Псевдоклассы

Псевдоклассы :hover и :active используются в CSS для стилизации элементов в зависимости от их состояния, что помогает улучшить взаимодействие с пользователем.

:active — это псевдокласс, который применяется в момент нажатия на элемент. Он позволяет создавать эффекты, которые делают интерфейс более отзывчивым. Например, изменение размера кнопки или ее цвета при нажатии помогает пользователю понять, что действие было зарегистрировано. В вашем примере:

:hover — это псевдокласс, который применяется к элементу, когда курсор мыши находится над ним. Это состояние позволяет разработчикам изменять стиль элемента, чтобы визуально обозначить, что он интерактивен. Например, можно изменить цвет фона, размер текста или добавить тень, чтобы кнопка выглядела более привлекательно. Пример использования:

.animated-button:hover {
    background-color: #FF00E5;
}

.animated-button:active {
    transform: scale(0.97);
}

Здесь кнопка уменьшится в размере на 3% при нажатии, изменяется задний фон, что создаст эффект "нажатия". Это делает взаимодействие с кнопкой более интуитивным и приятным для пользователя.
