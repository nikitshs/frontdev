6. Адаптивность

Адаптинвая вёрстка

Адаптивная верстка - это способ создания веб-сайтов, при котором сайт автоматически корректируется для оптимального отображения на различных устройствах благодаря динамической подгонке элементов под разные разрешения экранов.

Проще говоря, адаптивность - это адаптация блоков сайта под разные экраны, а не просто уменьшенное отображение сайта.

Пример адаптивности на различных устройствах

Способы адаптации сайта

Адаптация веб-сайтов под разные устройства — ключевой аспект современной веб-разработки. Есть несколько подходов и технологий, позволяющих создать адаптивный дизайн. Вот основные из них:

1. Отзывчивый (Responsive) Дизайн
• Медиа-запросы (Media Queries): CSS медиа-запросы позволяют применять различные стили в зависимости от характеристик устройства, таких как ширина экрана, высота, разрешение и т.д. Это основной инструмент для создания отзывчивых сайтов. Пример:
<style>
@media (max-width: 600px) {
        .sidebar {
                    display: none;
        }
}
</style>
Этот медиа-запрос скрывает боковую панель на устройствах с шириной экрана до 600 пикселей.
• Флексбокс (Flexbox): CSS Flexbox обеспечивает более гибкую раскладку, адаптируясь под размеры контейнера. Это особенно полезно для разработки адаптивных интерфейсов.
• CSS Grid: Подобно Flexbox, но предоставляет еще большие возможности для создания сложных макетов, которые легко адаптируются под разные размеры экранов.

2. Мобильная Версия Сайта
• Адаптивный дизайн через отдельный URL (m.example.com): Создание отдельной мобильной версии сайта, которая размещается на поддомене. Этот подход раньше был более популярен, но теперь уступает место отзывчивому дизайну из-за издержек на поддержку двух версий сайта.

3. Адаптивные Фреймворки
• Bootstrap 5: Популярный CSS-фреймворк, который предоставляет готовые компоненты и классы для создания адаптивных сайтов. Bootstrap использует медиа-запросы для адаптации макетов под различные размеры экранов и устройств. С его помощью можно легко достичь отзывчивости без необходимости писать много кастомного кода.
• Пример использования Bootstrap для создания адаптивного макета:
<div class="container">
    <div class="row">
        <div class="col-sm-12 col-md-8"> Основной контент </div>
        <div class="col-sm-12 col-md-4"> Боковая панель </div>
    </div>
</div>
Здесь .col-sm-12 делает элемент блочным на маленьких экранах, а .col-md-8 и .col-md-4 определяют ширину на средних и больших экранах.
• Tailwind CSS — это утилитарный CSS-фреймворк, который предлагает другой подход к созданию адаптивных дизайнов по сравнению с такими фреймворками, как Bootstrap. Вместо классического подхода с использованием предопределённых компонентов, Tailwind предоставляет набор утилитарных классов, которые можно применять непосредственно в HTML для стилизации элементов. Это позволяет разработчикам более тонко настраивать дизайн и адаптивность сайта, не покидая HTML-код. Пример использования Tailwind CSS для адаптивности:
<div class="text-sm md:text-base lg:text-lg xl:text-xl"> Этот текст изменит свой размер в зависимости от размера экрана. </div>
В этом примере текст будет адаптироваться к размеру экрана, увеличиваясь от `text-sm` для маленьких экранов до `text-xl` для экстра-больших экранов.
