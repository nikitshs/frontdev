Для начала зайдём в Visual Studio Code и создадим новую папку, в которой будет наш сайт, затем нажмём на иконку листа с плюсиком и создадим два новых файла index.html для разметки и style.css для стилей.

Далее, в файле index.html с помощью сочетания клавиш Shift + 1 создадим структуру Html документа. Поменяем название в теге <title> на “Мой первый сайт” и подключим файл с нашими стилями с помощью тега <link>. Затем зададим базовую структура из header, main, footer.

Переходим в файл style.css. В нём пишем *, это означает, что все стили, написанные в фигурных скобках будут применяться к каждому элементу на странице. Не забываем прописать логику анимации, чтобы они работали.

Но это еще не всё, ведь нужно этот сайт адаптировать, чтобы он корректно отображался на всех устройствах. Сейчас он будет правильно отображаться только на ПК. Для адаптации сайта мы будем использовать @media запросы. Пишем @media(max-width: 850px) - в скобках указываем от какой ширины будут приниматься стили прописанные внутри @media запросы, если ширина будет больше, то будут работать основные стили(которые вы прописали до этого, вне @media запроса). Важно уточнить, что @media запросы обязательно писать в самом низу css файла, иначе они не будут работать.

Затем, чтобы посмотреть свой сайт, устанавливаем расширение Live Server и в правом нижнем углу экрана нажимаем кнопку Go Live. При этом вы должны находиться в файле index.html. Ваш сайт откроется в браузере.
